# 这是DIXILOG的README

## 项目概述

基于MkDocs+Material主题构建的个人知识库与博客系统，部署于GitHub Pages

```bash
https://dixilog.github.io/
```

## 核心特性

- **Material主题定制**  
  深色/浅色双模式 | 响应式布局 | 中文优化 | 文楷字体支持
- **内容架构**  
  多维度知识体系（学习笔记/技术博客/阅读报告/资源整理）
- **增强功能**  
  - 搜索建议与高亮
  - 代码复制/注解功能
  - 图片灯箱效果
  - 文章阅读统计（字数/时长）
- **技术集成**  
  - 数学公式支持（KaTeX+MathJax）
  - Mermaid流程图
  - 时间轴追踪（创建/更新日期）
  - Google Analytics统计
