## 信息收纳

- 对于即时性强（如邮件、报名/候补通知）的信息，不妨巧用浏览器自带的<span style="border-bottom: 1.5px dashed orange;">分组功能</span>  
- 若希望系统性整理推免信息，则表格更合适，而飞书的<span style="border-bottom: 1.5px dashed orange;">多维表格</span>是一个不错的选择  


### 浏览器分组

以 Chrome 浏览器为例，关于标签分组的介绍可参考[这一篇博文](https://blog.csdn.net/hzj_java/article/details/113408331)

!!! success "怎么做"

    - 把你的<span style="border-bottom: 1.5px dashed orange;">所有邮箱</span>分为一个组，再把所有<span style="border-bottom: 1.5px dashed orange;">目标院校招生网页</span>分为一个组
     - 当你想要查询 email ，或者希望快速浏览一遍所有目标院校的招生网页时，只需打开相应的分组即可

!!! example ""
    
    ![example](https://cdn.jsdelivr.net/gh/dixiLOG/blogStatic/image-20251213162317291.png "分组效果")

> 当然你也可以选择收藏分组，见仁见智。但我很喜欢这种一键展开的打开方式，爽😝

[多维表格](https://base.feishu.cn/)以其<span style="border-bottom: 1.5px dashed orange;">表格不能只是表格</span>的特点成功让我做到所有资料 ALLinOne

## 报名材料准备

!!! question "都说要尽早准备，到底准备啥呢？"

    一般可分为<span style="border-bottom: 1.5px dashed orange;">报名材料、文书和面试准备</span>三个部分

本小节将罗列市面上常见的报名材料，一般够用，无非是各种排列组合（最后都要合并成一个PDF上交）

1. <span style="border-bottom: 1.5px dashed orange;">你的一寸帅照</span>

- 如果你没有近期照片的话，提前（寒假结束后）去海马体等照相馆选证件照套餐

- 拿到电子稿后裁剪成150*200，大小不超过50KB，`.jpg`和`.png`格式各准备一份

2. <span style="border-bottom: 1.5px dashed orange;">身份证、学生证扫描件</span>

- 身份证正反面，扫在一面上  
- 学生证带照片那一页，扫在一面上

3. <span style="border-bottom: 1.5px dashed orange;">本科院校信息</span>

- 通讯地址： 浙江省宁波市镇海区庄市街道天圣路488号宁波大学北校区学生二村

- 学校地址：浙江省宁波市江北区风华路818号

- 邮编：315211

- 专业代码：080703 通信工程（通创和普通班一致）

4. <span style="border-bottom: 1.5px dashed orange;">成绩总表、四六级证明、奖学金证明、荣誉奖证明、学籍证明、无处分证明</span>

- 风华宁大云证明处下载，这一点 nbu 值得夸夸😘  

> 但是下载的PDF带密，需要预处理，否则合并PDF时可能出现掉章的现象

5. <span style="border-bottom: 1.5px dashed orange;">学信网学籍在线验证报告</span>

进[学信网](https://my.chsi.com.cn/archive/bab/index.action)下载

6. <span style="border-bottom: 1.5px dashed orange;">手写签名</span>

- PNG图片去背景。不知道啥时候就用到了

7. <span style="border-bottom: 1.5px dashed orange;">==排名证明、院校申请表==</span>

- 最劝退的部分，因为需要盖章（需看清要求的是<span style="border-bottom: 1.5px dashed orange;">院教务办、校教务处、学院还是院党总支</span>的章）
  一般放暑假后教务办老师一周来一次，教务处在确定名额之前理论上是不给盖章的
- 不难注意到夏令营的报名时间基本都在放暑假之前，预推免则基本上在开学之后  
  所以别等到暑假才准备材料，也可以早些拿着排名证明去盖章
- 申请表千校千面，总体遵循 <span style="border-bottom: 1.5px dashed orange;">基本信息 → 教育背景  → 报名专业意愿 → 奖项罗列与个人陈述 → 签名盖章</span>

!!! warning "明年开始，暑假前就确定名额推免名额了，目标院校可能都会要求前六学期成绩"

> 排名证明模板可以去教务办要，或者点击下载：[学院落款](https://pan.baidu.com/s/1ieQmwWt-BnIWNMRMhjrJsw?pwd=3nh2)、[教务办落款](https://pan.baidu.com/s/1DgVqxXxN-8n81Vvon_glaw?pwd=jtcq)（在此鸣谢 ywt 老师）

8. <span style="border-bottom: 1.5px dashed orange;">获奖证书</span>

- 按竞赛级别降序扫描成一个PDF，最好是学科性专业竞赛（电赛、ACM类、数学建模、机器人等），校级及以下不建议纳入

9.  <span style="border-bottom: 1.5px dashed orange;">paper见刊、专利书</span>

- 没有，不懂


!!! tip "报名时还可能涉及的个人陈述放在<span style="border-bottom: 1.5px dashed orange;">文书与面试准备</span>那一节"
