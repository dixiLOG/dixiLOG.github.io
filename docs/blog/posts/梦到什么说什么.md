# 梦到什么说什么

  <div id="progress-bar"></div>
</div>

从前有座山，山里有座庙，庙里有两个和尚

老和尚问小和尚，你在干什么啊  
小和尚拿着尚在加工的枪管说，我在拉直膛线呀  
老和尚怒发冲冠，大胆，你也敢直呼圣上的名字  

小和尚哆哆嗦嗦，但不知所云  
老和尚见他满脸委屈，脸色稍缓和了一些。把头探出屋去，大雪漫山下人迹罕见，只有远处的草龙叠着连绵的山峦若隐若现

当今的圣上，直膛线，是远近闻名的暴君——暴饮暴食

<!-- more -->

有一次他听说世界是圆的，勃然大怒，命令大臣广仲找来了世界地图，当着朝臣的面吃起来。但世界地图太大了，直膛线没吃完，纸质太好又嚼不烂，只能吐出来

威严扫地，圣上责令广仲七步内把布鲁诺找来，活要见人死要见尸

广仲是他家乡那片原野上少有的开智猿人，故花名野原广智，他的乡亲们往往静悄悄的  
博闻强识如他，怎不知道布鲁诺在几百万年前就烧成灰了，还哪里找得到尸体呢  

为了活命，他掏出一条五步蛇来，边走边思索。直到被一座大山挡住，广仲才抬头发现了不远处不停下蛋的老母鸡愚公  
他问愚公可以做到吗，老翁淡然一笑，子子孙孙无穷匮也  
他终于停下了脚步，因为五步蛇咬不动了

圣上被他的真心打动，决定放宽时限  
广志毅然回家闭门造车。一天晚上，他正被关键技术勒得面如猪肝，一个毛桃坐着葫芦从天而降，告诉他把大象装进冰箱需要三步

梦醒了，他顾不得倦乏，那泥搓了一个形状特异的容器，并将其命名为纳尼  
人体实验之前先需要动物实验，广仲把愚公抓来放在纳尼上，拨动拉杆，一阵波动下，老母鸡竟原地消失了  
后世为了纪念愚公，又将纳尼命名为，时光机  

做好充分的思想准备，在静悄悄的亲友们的注视下，广仲也踏上了时光旅行  
但当他意识到自己是在前往未来而非过去时，一切都来不及了，因为d档下只有零速才能换r档  
眼一闭心一横，广志把时光机停了下来，却仍然看见了他的亲友们，远处的草龙，一切如故  

由于广志发现了时间是圆的，圣上特别赦免了他的死罪，还颁发了诺贝欠数学奖

那一天，孙悟空又被唐僧唠叨一宿，满肚子恼火，开着济公的车出门兜风散心  
到了一片草地上，他出神地望着星光闪闪，百无聊赖的拨弄着野草  
感受到一只蚂蚁趴在身旁，悟空便打趣他，问他把大象装进冰箱需要几步。蚂蚁梦醒了，哪里还有悟空的身影
