## 写在前面

记不住这么多逻辑学概念，于是基于自己的划线与批注，整理成一份 <span style="border-bottom: 1.5px dashed orange;">CHEATSHEET</span>

只摘出概念骨架和容易混淆的边界，若要理解论证的来处与例外，仍应回到原著

!!! note "术语约定"
    
    本页将 <font color="#e76f51">真 / 假</font> 用于语句、命题是否符合事实，  
    将 <font color="#2a9d8f">有效 / 无效</font> 用于前提能否必然推出结论（原书常用“对 / 错”）

**Key Point**：

1. 逻辑关心的是 <font color="#342521">推论形式是否必然有效</font>；这和语句的真伪属于不同层次
2. 讨论的分歧未必来自观点本身，也可能来自 <font color="#2a9d8f">语言功能、条件范围或概念边界</font>
3. 逻辑训练不保证真结论，却要求前提、条件与推出关系可以逐一交代，不让情绪、成见、利害或流行意见替代论证

## 绪论

### 逻辑

* 定义：逻辑是<span style="border-bottom: 1.5px dashed orange;">必然有效的推论规律的科学</span>  
> 它研究的不是某个结论是否合我心意，而是 <font color="#e76f51">在前提既定时，结论是否非得如此不可</font>  
* 既往经验能帮助人迅速行动，但经验形成的预设也会在陌生情境中遮蔽新证据，逻辑便起到泛化的作用  


### 真假与有效

* <font color="#e76f51">语句</font>可以真或假；<font color="#2a9d8f">推论</font>可以有效或无效。二者相互独立，不能互相替代
* 只有在<span style="border-bottom: 1.5px dashed orange;">前提为真且推论有效</span>时，结论才必定为真
* 无效推论不等于结论必假。它只说明结论没有被前提保证；结论可能碰巧为真，也可能为假

<center>

| 前提 | 推论形式 | 结论的真值 |
| --- | --- | --- |
| 真 | 有效 | 必真 |
| 假 | 有效 | 不定 |
| 真或假 | 无效 | 不定，不能由前提保证 |

</center>


```text title="无效推论：结论事实上为假"
前提一：所有柠檬都是酸的  
前提二：没有橘子是柠檬  
结论：没有橘子是酸的
```


> 结论在事实上是假的，但更关键的是：它<span style="border-bottom: 1.5px dashed orange;">并未由前提推出</span>

```text title="无效推论：结论碰巧为真"
前提一：所有柠檬都是酸的  
前提二：没有西瓜是柠檬  
结论：没有西瓜是酸的
```

> 这组推论的形式同样无效；只是结论恰好为真  
> 不能因为结论符合经验，就倒过来把推论当成有效（见三段式有效性判断）

!!! warning "三个判断层次"
    
    * “结论是真的”是事实判断  
    * “结论由前提推出”是逻辑判断  
    * “前提可靠”是证据判断



### 推论

* 推论是获得知识的间接方式之一  
* 猜测提供假设，观察与经验提供材料，推论说明<span style="border-bottom: 1.5px dashed orange;">为什么材料足以支持结论</span>
* 推理（reasoning）是一次具体的思考活动；推论强调其中可抽离、可重复检验的型式  
  > 换句话说，逻辑研究的不是每一个具体前提，而是各种推理共同依据的型式
* 有效推论意味着：只要前提为真，结论就不可能为假，即前提<span style="border-bottom: 1.5px dashed orange;">涵蕴</span>结论
* 逻辑和数学都重视形式、符号与演算，彼此关系很近；但不宜直接把逻辑等同于数学
* 常见的推论范式有：选取推论、条件推论、三段式

***

## 常见推论

> 语言的功能很多：抒情、命令、安慰、煽动、表达情绪  
> 这里讨论的是能够承担真假、是非或对错判断的“讨论语言”

### 选取推论

* 基本形式：`Y 是 φ 或者 Y 是 θ`  
* 检查一个选取式能否支撑后续结论，要分别问：选项能否同时成立（相容性），以及它们是否覆盖所有可能（穷尽性）
     * <font color="#2a9d8f">相容</font>表示选项可以有交集，并非必然二选一；<font color="#e76f51">互斥</font>才表示不能同时成立  
     * <font color="#2a9d8f">穷尽</font>表示选项的并集覆盖讨论范围；只有两个选项时，它们才可理解为互为补集

!!! quote "dixi 有话说"
    
    很多网络讨论表面上像在争观点，实际上一方在表达立场，一方在表达情绪，另一方才在试图讨论真假。先确认彼此是否在做同一种语言行为，往往比立刻反驳更重要

### 条件推论

> 浙江高考数学选填常客

* 基本形式：若 <code>P</code>（前件，antecedent），则 <code>Q</code>（后件，consequent），记作 <code>P → Q</code>；<code>¬P</code> 表示“非 P”
* <code>P → Q</code> 表示：<code>P</code> 是 <code>Q</code> 的<span style="border-bottom: 1.5px dashed orange;">充分条件</span>，<code>Q</code> 是 <code>P</code> 的<span style="border-bottom: 1.5px dashed orange;">必要条件</span>，但不能据此反推 <code>Q → P</code>
* 肯定前件可以肯定后件；否定后件可以否定前件。否定前件或肯定后件，均不足以推出另一端的真假

<center>

| 已知 | 可推出 | 常见误用 |
| --- | --- | --- |
| P 真 | Q 真 | 无 |
| P 假 | Q 不定 | 否定前件 |
| Q 假 | P 假 | 无 |
| Q 真 | P 不定 | 肯定后件 |

</center>

* 若 <code>P ↔ Q</code>，即前件既充分又必要，才可在四个方向上作相应双射

!!! quote "dixi 有话说"
    
    逻辑在捍卫模糊性的理所应当：不能确定时，就让结论停在“不能确定”，而非用直觉替它补上答案

### 二难式

* 二难式（dilemma）是由选取推论和条件推论复合而成的论证形式，<font color="#e76f51">并不天然是诡辩</font>。选项不穷尽、条件不成立或前后偷换条件时，它才会变成“假两难”
* 其基本形式如下：

<center>

| 形式 | 结构 |
| --- | --- |
| 简单肯定前件 | 甲 → 卯；乙 → 卯；甲 或 乙；所以 卯 |
| 简单否定后件 | 甲 → 卯；甲 → 未；¬卯 和 ¬未；所以 ¬甲 |
| 复合肯定前件 | 甲 → 乙；丙 → 丁；甲 或 丙；所以 乙 或 丁 |
| 复合否定后件 | 甲 → 乙；丙 → 丁；¬乙 和 ¬丁；所以 ¬甲 和 ¬丙 |

</center>

!!! note "反驳方式"
     第一，否认前件为后件的充分条件  
     第二，否认选取语句是互不相容或是共同穷尽的

## 语句和类

* 不分析 P、Q 各自代表什么，只讨论“由 P 能否推出 Q”的，是外部推论（outward inference）
* 若进入语句内部，分析词项的普及、主宾位置和类的关系，则是内部推论（inward inference）
* 逻辑在这里主要处理能够判定真假的<span style="border-bottom: 1.5px dashed orange;">陈述句</span>
* 主宾词式可分析为词主、词系和词宾，主语和宾语为词端  
* 词端可以指一个类，也可以指个体；若宾位词端是形容词，逻辑上可把它当作一类来处理

<span style="border-bottom: 1.5px dashed orange;">主宾词式的语句可分为四种：</span>

* 全谓肯定（A）：所有 S 是 P
* 全谓否定（E）：没有 S 是 P
* 偏谓肯定（I）：有些 S 是 P
* 偏谓否定（O）：有些 S 不是 P

<center>

| 句式 | 主词是否普及 | 宾词是否普及 |
| --- | --- | --- |
| A | 是 | 否 |
| E | 是 | 是 |
| I | 否 | 否 |
| O | 否 | 是 |

</center>

> 普及与否不是词本身的属性，而是词项在这个句式中被谈到了多大范围  
> 这是判断三段式能否成立的基础

***


## 位换和质换

* 位换（conversion）是主、宾词端互换位置；质换（obversion）则改变语句的肯否，同时把宾词换成它的补类，使新语句与原语句等值
* 全量位换可用于 E、I 式：没有 S 是 P，可换为没有 P 是 S；有些 S 是 P，可换为有些 P 是 S
* A 式只能作限量位换：所有 S 是 P，可推出有些 P 是 S。这里沿用传统三段论的存在预设，即 S 类确有成员
* O 式不能作有效位换；把“有些 S 不是 P”换成“有些 P 不是 S”会改变原意
* 质换四式：SAP → SE¬P；SEP → SA¬P；SIP → SO¬P；SOP → SI¬P

## 对待关系

* 对待（opposition）讨论同一主词、宾词下各语句的真假约束

<center>

| 关系 | 对应句式 | 真假约束 |
| --- | --- | --- |
| 反对 | others | 真则假，假则真假不定 |
| 等差（从属） | A / I，E / O | 全谓真可推偏谓真；偏谓假可推全谓假；其余方向不定 |
| 独立 | I / O | 真假不定 |

</center>


***

## 三段式

* 定言三段式（categorical syllogism）的成素限于 A、E、I、O 四种主宾词式，有两个前提和一个结论  
* 大前提并非“排在第一句的前提”，而是包含<span style="border-bottom: 1.5px dashed orange;">大词</span>的前提；小前提包含小词  
* 词项有三个：小词 S、大词 P、共词（中项）M  
* M 出现在两个前提中，不出现在结论中，负责把 S 和 P 接起来  
* 小词一定出现在结论的主位词端，大词一定出现在宾位词端

三段式的有效性判断：

* 共词 M 至少普及一次，否则两个前提可能仅涉及 M 的不同部分

    ```text title="共词不普及"
    你爸爸是人
    我也是人
    所以我是你爸爸
    ```

* 大词 P 与小词 S 凡在前提中没有普及，结论中也不得普及；否则结论谈论的范围比前提更大

    ```text title="大词不当周延"
    所有柠檬都是酸的
    没有橘子是柠檬
    所以没有橘子是酸的
    ```

    “酸的”在第一前提中未普及，却在否定结论中被说成全部酸的东西，属于<span style="border-bottom: 1.5px dashed orange;">大词不当周延</span>

* 两个前提都是否定语句，不能得出结论
* 两个前提中若有一个是否定语句，结论必须是否定；两个前提都肯定时，结论必须肯定

## 关系

* 关系是联系对象的方式，用 R 表示。谈论 aRb 时，重点不是 a、b 各自是什么，而是 R 具有什么结构性质
* 基本性质有三种：自反性（reflexivity）、对称性（symmetry）和传达性（transitivity，现代常译为“传递性”）  

!!! example "示例"
     自反性：对任一对象都有 xRx，如 “等于”“相似”  
     对称性：若 aRb，则 bRa，如 “同年”“坐在对面”  
     > 准对称性：若 aRb ，则 aRb 或者不是 aRb，如“我拿你当兄弟”  
     
     传达性：若 aRb 且 bRc，则 aRc，如 “包含”  
     > “=”：既自反、又对称、又有传达性  

***


## 思想三律

* 同一律（Law of Identity）：语意原则。一个符号在同一次论证中应保持同一指代或意义；否则即使句子相同，也无从分析。可简记为 A = A

    ```text title="语意滑移"
    “道可道，非常道”中的两个“道”若不先辨明意义，不能直接套用同一律
    “抓鲁迅和我周树人”什么关系，也是在借同名、异称制造语意滑移
    ```

* 矛盾律（Law of Contradiction）：语法原则。互为否定的 A 与 ¬A 不能同真；在经典二值逻辑中，它们也不能同假  
* 排中律（Law of Excluded Middle）：语意原则。对于意义确定的命题，A 与 ¬A 之间没有第三个真值，二者必有一真  
* 三律要求的是最低限度的语意稳定与推论一致性

## 语意界说

界说（definition）就是用既有符号来规定或陈示某个新符号的用法与意义

* 语法界说 / 名目界说：主要介绍新名词、新记号或构造规则，不直接说明其所指意义，如阶乘的定义式
* 语意界说：界定端陈示被界定端的意义，可以用来消除歧义、调节意义范围、增添新意义或简写表达

```text title="语意界说，比如"
逻辑是必然有效的推论之科学
```

定立界说时需遵守：

* 非必要不另造新名；确有新对象或新区别时再命名
* 在<span style="border-bottom: 1.5px dashed orange;">同一讨论脉络</span>中不要悄悄改变一个名词的定义
* 定义应当清楚，范围不宜过宽或过窄，不以被定义词本身循环解释自己
* 能用肯定方式说明时优先用肯定方式

***

## 分类与归类

* 知识从经验开始；要让经验可处理、可传递、可积累，就必须经过分类与归类
* 分类：把一个已知的类按共同标准划分为次类，是制定规则、搭建层级的过程
* 归类：依据某一对象的性质或共同点，把它放入相应的类，是把具体对象应用到规则中的过程
* 好的分类应尽量保持同一标准、层级相当，并在任务所需范围内避免重叠和遗漏

***

## 诡论与谬误

```text title="诡论引子"
“一切真理是相对的”
“理发师悖论”
“这句话是假的”
```

<span style="border-bottom: 1.5px dashed orange;">语言一旦自我指涉、层次混用或定义滑动，就可能制造看似无解的冲突</span>

* 诡论（paradox）是看似可接受的推演却导向冲突或荒谬的情形；语言的自我指涉是其中一个重要来源，但并非所有诡论都只由自指造成
* 谬误（fallacy）是论证在形式、语意或关联上出了问题。依原书的梳理，可注意三类：<font color="#e76f51">型式的谬误</font>、<font color="#e76f51">语意的谬误</font>、<font color="#e76f51">不相干的谬误</font>
* 区分 logical、non-logical 与 illogical：有些表达无所谓合不合逻辑，这不同于违反逻辑


!!! quote "dixi 有话说"
     先辨认语言在做什么，才知道该不该用逻辑去审查它  
     所以写作、演讲、辩论、宣传、安慰 等，不必然合乎逻辑
