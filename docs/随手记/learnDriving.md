### 一、报名与体检

1. **条件**：年满 18 周岁，身体条件符合规定
2. **体检**：到指定医院，重点检查视力、四肢活动度、听力等
3. **报名**：身份证、照片、体检表交驾校 → 驾校录入系统 → 车管所建档
4. **学员证**：审核通过后，拿到学习驾驶证明

### 二、科目一（理论）

- **内容**：交通法规、道路安全、基础知识
- **题型**：100 题，单选/判断，45 分钟
- **合格标准**：90 分及以
- **补考规则**：一次补考机会，两次都没过 → 重新预约

![](https://cdn.jsdelivr.net/gh/dixiLOG/blogStatic/TAkNbYbl5oBZ7zxCzwRcAjBSn8r.png)

### 三、科目二（场地）

C2 自动挡考试项目和 C1 相同，但没有“离合控制”要求，重点在刹车和方向掌控

1. **倒车入库**

   - 自动挡不能“半联动”，靠稳油/刹车控制速度
   - 要点：保持车速匀稳，打方向果断

2. **坡道定点停车与起步**

   - 自动挡靠刹车控制，起步时松刹车即可，不会后溜
   - 要点：前保险杠停在黄线以内，停车后立即挂 D 挡起步

3. **侧方停车**

   - 控制车身与边线 ≤30cm
   - 注意不能中途停车过久，否则判不合格

4. **曲线行驶**

   - 保持低速，方向控制到位

5. **直角转弯**

   - 转弯要提前打方向，避免压线

**扣分点**

- 压线 / 碰杆：立即不合格
- 车速过快：超速判不合格
- 中途停车过久：视为考试失败

![](https://cdn.jsdelivr.net/gh/dixiLOG/blogStatic/N7kmbmpbSoS9OnxmWLGcp5Xfnix.png)

### 四、科目三（夜间灯光模拟 + 实路 ）

### （一）模拟夜间灯光考试

- **考试方式**：上车后，语音播报灯光指令，学员操作灯光。
- **常见指令**：

  - 夜间起步 → 开近光灯。
  - 通过无信号路口 → 远近光交替。
  - 遇对向来车 → 关远光，开近光。
  - 临时停车 → 开示宽灯 + 双闪。
- **要求**：操作准确、迅速，错一次就可能直接挂掉。

### （二）道路驾驶技能考试

C2 自动挡不需要换挡操作，重点在安全文明驾驶习惯。

**必考项目**：

1. **上车准备**：绕车一周 → 系安全带 → 调整座椅/后视镜 → 点火
2. **起步**：打左转向灯，观察后视镜和周围情况
3. **直线行驶**：双手握方向盘，保持车身稳定
4. **变更车道**：打灯 → 观察后视镜 → 控制方向
5. **超车 / 会车**：注意打灯和观察
6. **通过路口**：遵守信号灯，注意避让
7. **掉头**：选择允许掉头的地点，打灯
8. **靠边停车**：靠边 ≤30cm，挂 P 挡，拉手刹

**扣分点（C2 关键）**

- 未打转向灯或打灯不及时（<3 秒）→ 扣 10 分或直接不合格
- 不观察后视镜 / 不回头观察盲区 → 扣 10 分
- 不按规定让行 → 扣 100 分直接挂科
- 直线行驶压线、随意变道 → 直接挂科

### 五、科目四

- **内容**：文明驾驶、事故处理、恶劣天气驾驶等
- **题型**：50 题（单选、多选、判断）
- **合格标准**：90 分及以上


![](https://cdn.jsdelivr.net/gh/dixiLOG/blogStatic/TQl8bvPlEowCuOxuYBjcxUpInNe.png)

---

### 六、领取驾驶证

- 科目四合格后，领取驾驶证
- **实习期**：1 年。需张贴实习标志，不得单独上高速

---

### 总结

宁波 C2 驾考流程为：
**报名 → 科目一 → 科目二（场地） → 科目三（灯光 + 道路） → 科目四 → 领证**

- 科二重在**车速控制和方向精准**
- 科三重在**灯光模拟 + 安全驾驶习惯**
- 自动挡省去了换挡操作，但对“观察 + 打灯”的要求更严格

### 练车

#### 第一次

时间：11.4

练习时长：2h

内容：

- 左倒车入库：
  - 从停车位开始，松刹车至半车出库，`左打一圈+90度` 保持，直至压线
  - 停车挂倒档，观察左后视镜，车尾遮盖库右角时打死，<u>降速</u>观察与左边线间距（这一步是重点）
  - 若车尾快要遮住左角（距离一鞋子）则改为打一圈，否则直到车身到与左线平行（在后视镜中，<u>平行时</u>车尾离线距离比车身略窄一些）快速回正，前边线离后视镜一指时停车
- 右倒车入库：
  - 从停车位开始，松刹车至半车出库，右边打死，目视前方，车身 45° 角左右回为一圈，垂直后回正，压线停车
  - 停车挂倒档，观察左后视镜，出现库左角时降速回正，直至车把手内侧与内边线相交，再打满直至平行快速回正，前边线离后视镜一指时停车

  > 这里如果出库打多了，倒车时基本最后才看见角落，这时候就一直打着


- 问题与待改进建议：

  - 打方向盘要果断而不乱，特别是最后入库的微调
  - 速度降下来，学会保持怠速
  - 灵活一些，重点是掌握车感，从有规矩到无规矩

### 练车


!!! tip "要点"

    - 提前熟记路线很重要，优雅提前打灯
    - 先看后视镜 → 打灯 → 动方向盘
    - 没语音提示可以不打灯，打灯必须先硬控 `3` 秒
    - 进路口不要压线
    - 眼看远，手放松，身坐正，车不歪
    - 一般车速在 25~30，加速路段保持 40 以上 5 秒
