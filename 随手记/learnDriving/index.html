    学车记¶

     

    约 4006 个字 • 83 行代码 预计阅读时间 23 分钟

    绪论¶

    再次化身云玩家，通过人家的文字大致了解学车的整体流程

    Background¶

    • 宁大鹭子渠道（天辰驾校）
    • C2
    • 宁波横溪考场

    一、报名与体检¶

    1. 条件：年满 18 周岁，身体条件符合规定
    2. 体检：到指定医院，重点检查视力、四肢活动度、听力等
    3. 报名：身份证、照片、体检表交驾校 → 驾校录入系统 → 车管所建档
    4. 学员证：审核通过后，拿到学习驾驶证明

    二、科目一（理论）¶

    • 内容：交通法规、道路安全、基础知识
    • 题型：100 题，单选/判断，45 分钟
    • 合格标准：90 分及以
    • 补考规则：一次补考机会，两次都没过 → 重新预约

    三、科目二（场地）¶

    C2 自动挡考试项目和 C1 相同，但没有“离合控制”要求，重点在刹车和方向掌控

    1. 倒车入库
    • 自动挡不能“半联动”，靠稳油/刹车控制速度
    • 要点：保持车速匀稳，打方向果断
    1. 坡道定点停车与起步
    • 自动挡靠刹车控制，起步时松刹车即可，不会后溜
    • 要点：前保险杠停在黄线以内，停车后立即挂 D 挡起步
    1. 侧方停车
    • 控制车身与边线 ≤30cm
    • 注意不能中途停车过久，否则判不合格
    1. 曲线行驶
    • 保持低速，方向控制到位
    1. 直角转弯
    • 转弯要提前打方向，避免压线

    扣分点

    • 压线 / 碰杆：立即不合格
    • 车速过快：超速判不合格
    • 中途停车过久：视为考试失败

    四、科目三（夜间灯光模拟 + 实路 ）¶

    （一）模拟夜间灯光考试¶

    • 考试方式：上车后，语音播报灯光指令，学员操作灯光。

    • 常见指令：

    • 夜间起步 → 开近光灯。

    • 通过无信号路口 → 远近光交替。
    • 遇对向来车 → 关远光，开近光。
    • 临时停车 → 开示宽灯 + 双闪。
    • 要求：操作准确、迅速，错一次就可能直接挂掉。

    （二）道路驾驶技能考试¶

    C2 自动挡不需要换挡操作，重点在安全文明驾驶习惯。

    必考项目：

    1. 上车准备：绕车一周 → 系安全带 → 调整座椅/后视镜 → 点火
    2. 起步：打左转向灯，观察后视镜和周围情况
    3. 直线行驶：双手握方向盘，保持车身稳定
    4. 变更车道：打灯 → 观察后视镜 → 控制方向
    5. 超车 / 会车：注意打灯和观察
    6. 通过路口：遵守信号灯，注意避让
    7. 掉头：选择允许掉头的地点，打灯
    8. 靠边停车：靠边 ≤30cm，挂 P 挡，拉手刹

    扣分点（C2 关键）

    • 未打转向灯或打灯不及时（<3 秒）→ 扣 10 分或直接不合格
    • 不观察后视镜 / 不回头观察盲区 → 扣 10 分
    • 不按规定让行 → 扣 100 分直接挂科
    • 直线行驶压线、随意变道 → 直接挂科

    五、科目四¶

    • 内容：文明驾驶、事故处理、恶劣天气驾驶等
    • 题型：50 题（单选、多选、判断）
    • 合格标准：90 分及以上

    六、领取驾驶证¶

    • 科目四合格后，领取驾驶证
    • 实习期：1 年。需张贴实习标志，不得单独上高速

    总结¶

    宁波 C2 驾考流程为： 报名 → 科目一 → 科目二（场地） → 科目三（灯光 + 道路） → 科目四 → 领证

    • 科二重在**车速控制和方向精准**
    • 科三重在**灯光模拟 + 安全驾驶习惯**
    • 自动挡省去了换挡操作，但对“观察 + 打灯”的要求更严格

    流程¶

    报名¶

    给鹭子客服交 200 定金后，和对应驾校的老板约一下报名时间即可

    报名当天需要带身份证，先花 40 进自动体检机（和拍大头照的机子一样）体检

    接着填基本信息，签合同交尾款（自行留底），下载注册交管 12123（当天注册，不可提前），关注宁波交警公众号

    当天内会收到短信，大致如下，说明报名成功

    【宁波交警】dixi，您已成功报名初学（C2）机动车驾驶理论学习，学习编号：xxxxxxxxx，请联系驾校预约现场学习时间。

    课时学习¶

    • 理论学习扫码下载【驾考宝典】选择已报名—小车科目一学习

    • 微信关注公众号：宁波交警。选择业务服务 → 初学增驾 → 自助学习

    • 300 题做到 80 分即可（12 课时）

    • 安全警示教育视频 + 50 道题目（4 课时）+ 应知应会视频（6 课时）
    • 现场理论学习一天以短信时间为准（8 课时）

    公众号微信题目做完 + 镇海电大完成 1 天理论学习，在宁波交警里自查满 30 课时后联系驾校预约理论考试

    线下理论学习一天（四小时）。与后续的理论考试一样，地址都是镇海电大（考科三的地方也能直接考科四，不妨约在同一天）

    报名与考试¶

    在 交管12123 上预约科一时间，最早可约一周之后。若预约成功则在考试第五天 app 告知（考试前一天也会短信提醒）

    考试当天带身份证， 考场前先支付宝缴费（80），存包后排队刷身份证进候考闸机（这里就是签到，需要赶在你 所约时间的 ddl 前进去）

    和医院挂号一样，显示屏上弹出大名，即可进考场找到位置开始考试

    考试 UI 相当的简朴，会有摄像头捕捉，点击确定后当场判断正确与否，正确没反应，错误会弹窗

    总体难度上比驾考宝典简单不少，冲就完了

    考完后签字即可离场，后续联系驾校约车

    理论先行¶

    练车¶

    第一次¶

    时间：11.4

    练习时长：2h

    内容：

    • 左倒车入库：
    • 从停车位开始，松刹车至半车出库，左打一圈+90度 保持，直至压线
    • 停车挂倒档，观察左后视镜，车尾遮盖库右角时打死，降速观察与左边线间距（这一步是重点）
    • 若车尾快要遮住左角（距离一鞋子）则改为打一圈，否则直到车身到与左线平行（在后视镜中，平行时车尾离线距离比车身略窄一些）快速回正，前边线离后视镜一指时停车
    • 右倒车入库：
    • 从停车位开始，松刹车至半车出库，右边打死，目视前方，车身 45° 角左右回为一圈，垂直后回正，压线停车
    • 停车挂倒档，观察左后视镜，出现库左角时降速回正，直至车把手内侧与内边线相交，再打满直至平行快速回正，前边线离后视镜一指时停车

    这里如果出库打多了，倒车时基本最后才看见角落，这时候就一直打着

    • 问题与待改进建议：

    • 打方向盘要果断而不乱，特别是最后入库的微调

    • 速度降下来，学会保持怠速
    • 灵活一些，重点是掌握车感，从有规矩到无规矩

    第二次¶

    时间：11.5

    练习时长：2h

    内容：倒车入库

    改进与问题：

    • 更松弛，可几乎不借助地面标记倒库，已充分利用右后视镜
    • 车速控制仍不理想，有点飘
    • 右侧倒库仍有概率压线（出库打太多了，过于贴库）
    • 考试的时候还是要慢慢开 😂

    据说剩下的项目非常简单，倒库练完就差不多了

    第三次¶

    时间：11.11

    练习时长：2h

    内容：科二所有项目，打圈熟悉

    • 直角拐弯（左）：中间靠右，打左转向灯，脚与左边线相交时左打死（车速慢），转过去后回正
    • S 弯道：根据标志点分别左一圈、回正、右一圈，探头微调即可，前轮出 S 弯即可转向
    • 侧方停车：根据标志点停车，静止状态下右打一圈，挂 R 档，左后视镜见标志回正（不能停车），左后轮压线后快速左打死，直到停进去；方向盘保持，左转向灯挂 D 档，见标志回正，再右打一圈出（注意不能压线）

    半小时一遍过，自由活动了，月底考掉；有标志点简单不只一点点啊

    实地模拟¶

    • 时间：11.23
    • 地点：横溪翠竹科二考场

    点位分布与流程¶

    • 上车后调整座椅靠背后视镜，系安全带
    • 刷身份证，听语音提示开始

    自动挡无半坡起步，直接从最左侧通道上去，进直角拐弯通道，对齐红线处位置

    肩至红线处，减速左打满

    驶出后靠左道行驶，若排队则停车等待。下方红标处左打满，进入侧方停车

    回正后方向盘对齐红标（箭头右侧）

    慢行至红标处停车，右打一圈，挂 R 档

    后与无异

    注意轮胎可以微微多压一点再打

    打左转向灯驶出，注意不要压线

    靠左后走最靠墙车道，排队进 S 弯，方向盘对准缺口右侧，到标记处右打一圈，停车，直到前车彻底开出去

    下面是 S 弯的几处标记点，分别代表回正，左转一圈，回正

    可以略早打

    出弯后根据组别到 2/4/6 中的一个，方向盘对准箭头右尖尖，驶入

    右侧倒库，到红线处打方向；左侧倒库与联系时无异

    倒库完成后，正常出库即可，考试完成，返回出发点

    正式考试¶

    一切顺遂！

    • 时间：20251124 早上 8 点

    六点集合出发，一小时车程到达，再熟悉场地

    八点扫身份证进候考大厅，见进场（事实上第一批来的三十人都显示进场）后再刷第二次闸机进广场，等待叫号

    C1 和 C2 是分开的，8 点 50 轮到我，上一个哥们下车后我轮次上车

    调整好系安全带，放身份证，等语音叽里咕噜结束，正式开始

    因为车里没人，我也比较 relax，所以一直在哼歌 🤪

    按照模拟路径一路开，因为正好没车，几乎没有停顿

    分配到了 2 号倒库（车上会标），出 S 弯道后倒库

    这里的车型略有不同故还是需要自己靠感觉微调的

    右倒库十分顺利，左倒库有点打多了，差点几公分压到左边，不过正着开进去就 OK

    全程都没怎么听语音，肌肉记忆就完事了，反正不熄火随便造

    据说下个月就能拿证了，比预计的快了好几个月啊

    练车¶

    要点

    • 提前熟记路线很重要，优雅提前打灯
    • 先看后视镜 → 打灯 → 动方向盘
    • 没语音提示可以不打灯，打灯必须先硬控 3 秒
    • 进路口不要压线
    • 眼看远，手放松，身坐正，车不歪
    • 一般车速在 25~30，加速路段保持 40 以上 5 秒

    第一次¶

    • 时间：12.2
    • 练习时长：2h
    • 内容：科三基本操作（顺序不分先后）

    上车先调整座椅后视镜（相较于科二上抬一些，能看见后车），检查灯光归位

    被告知开始后下车关门绕一圈按按钮 再上车系安全带正式开始语音模拟灯光考试

    第二次¶

    • 时间：12.3
    • 练习时长：1h
    • 内容：科三基本操作，打圈

    一路聊天着就开完了，但还无法做到完全放松聊天

    第三次¶

    • 时间：12.5
    • 练习时长：2h
    • 内容：实际上路

    开到外面转了一圈，然后直接从植物园开回来了，好玩

    实地模拟¶

    • 时间：12.7
    • 练习时长：2h
    • 内容：场地熟悉

    比较简单，放松即可，灯光别错

    科三考试要点总结

    • 上车检查灯光复位
    • 起步后停车打灯
    • 靠虚不靠实
    • 超车变道停稳打右灯再给油
    • 掉头左打灯，注意电瓶车

    正式考试¶

    前面和科二一样的流程

    • 上车后调整完毕开始，走流程
    • 我抽到了二号线，脑子过了一遍路书
    • 按照路书走一遍，当时运气好九点多路上车很少
    • 侧边停车后自行到地方签字，紧接着考科四
    • 科四考完后（差点挂了 🤣）宣誓，拿证

    这里注意证件照要是希望用自己带的，需要在给身份证的时候就递上去 bro 过于老实，等我意识到时木已成舟
